## 📝 PRÓXIMOS PASSOS - INSTRUÇÕES MANUAIS

### PASSO 1: Fazer Upload do Ícone
1. Acesse: https://play.google.com/console/u/0/developers/app/4972652734342206533/app-store-listing
2. Na seção **"Ícone do aplicativo"**, clique em **"Adicionar recursos"**
3. Upload do arquivo: `expenseflow_icon.png` (512×512px, PNG)
4. Clique em **Salvar**

### PASSO 2: Fazer Upload da Feature Graphic
1. Na mesma página, na seção **"Recurso gráfico"**, clique em **"Adicionar recursos"**
2. Upload do arquivo: `expenseflow_feature.png` (1024×500px, PNG)
3. Clique em **Salvar**

### PASSO 3: Preencher Informações do App

```
Gerencie suas despesas com facilidade e controle
```

### PASSO 4: Adicionar Screenshots
Você precisará capturar prints de:
1. **Tela 1**: Dashboard (já tem a imagem)
2. **Tela 2**: Nova Despesa
3. **Tela 3**: Minhas Despesas
4. **Tela 4**: Aprovações
5. **Tela 5**: Relatórios (opcional)

**Requisitos dos Screenshots:**
- Tamanho: 1080×1920px ou similar
- Formato: PNG/JPEG
- Máximo: 8 imagens
- Mínimo: 2 imagens (recomendado 4-5)

### PASSO 5: Configurações Adicionais

#### Categoria
- Selecione: **Finanças**

#### Classificação de Conteúdo
- Marque a classificação apropriada (geralmente LIVRE ou 12+)

#### Privacidade
- Configure conforme sua política (coleta dados sensíveis?)

#### Permissões
- Revise as permissões necessárias do app

### PASSO 6: Revisar e Enviar para Análise

1. Vá para **"Visão Geral da Publicação"** (menu esquerdo)
2. Revise todos os detalhes:
   - ✓ Ícone
   - ✓ Feature Graphic
   - ✓ Descrição
   - ✓ Screenshots
   - ✓ Informações sobre o app
3. Clique em **"Enviar para Revisão"**
4. Leia os avisos de conformidade
5. Confirme o envio

## 🎨 Recursos Criados

### Filosofia de Design
A paleta visual segue o conceito de **"Fluxo Controlado"**:
- Azul profundo (#0A3A66) - confiança
- Azul médio (#1E5BA8) - profissionalismo  
- Ciano vibrante (#00B4D8) - dinamismo
- Branco/Cinza - clareza

### Elementos do Ícone
- Moeda com linhas de fluxo
- Símbolo $ central
- Design limpo e moderno
- Reconhecível em tamanhos pequenos

### Elementos da Feature Graphic
- Gradiente azul elegante
- Padrão de linhas de fluxo
- Moedas representando movimento
- Gráfico de barras para representar categorias

2. Use PNG para imagens (não JPEG para ícone)
3. Não deixe campos obrigatórios em branco
4. Revise a política de metadados do Google Play
